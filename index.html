⚙ First-Time Setup Instructions (click to expand)
  1. Create the SharePoint List — go to your SharePoint site → New → List → name it ATS Label Log.
    Add these columns (all "Single line of text"): WorkOrder, PartNumber, SerialNumber, Customer, Description, Condition, Quantity, QCBy.
  2. Register an Azure AD App — portal.azure.com → Azure Active Directory → App registrations → New registration.
    Name: ATS Label Printer • Type: Single-page application (SPA) • Redirect URI: the OneDrive/SharePoint URL of this file.
  3. Grant permission → API permissions → Add → Microsoft Graph → Delegated → Sites.ReadWrite.All → Grant admin consent.
  4. Paste the IDs from app Overview into the CONFIG block at the top of this file: clientId, tenantId, and your SharePoint site path.
⚠ CONFIG not set up — SharePoint logging is disabled. Labels can still be printed without it.
